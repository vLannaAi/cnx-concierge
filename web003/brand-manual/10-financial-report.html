Report Design Guidelines

Layout Structure

  • Standard A4 page size (210 × 297mm)
  • Minimum 25mm margins on all sides
  • Clear section breaks with consistent spacing
  • Header with wordmark and report title
  • Footer with company information

Typography Hierarchy

  • Report title: 2.5rem, CNX Navy, bold
  • Section headings: 1.8rem, CNX Navy, semi-bold
  • Body text: 1.05rem, Ink, regular, line-height 1.7
  • Table text: 0.95–1rem, Ink, regular
  • Footnotes: 0.85rem, Harbor Gray, italic

Table Design

  • Header row: CNX Navy text, light paper background
  • Borders: 1px solid, Harbor Gray for rows, 2px for headers
  • Alternating row colors optional (very subtle)
  • Right-align numerical data
  • Left-align text data

Chart Integration

  • Use palette colors: CNX Navy, Lavender, Harbor Gray
  • Maintain sufficient contrast for readability
  • Include clear labels and legends
  • Position charts near related tables
  • Ensure charts are print-friendly
