Icon & Profile Usage

Limited Application for Profile Pictures and Avatars

CNX Concierge maintains a typography-first identity with no symbols or icons in standard brand applications. However, Icon Positive (visual/png/Icon Positive.png) and Icon Negative (visual/png/Icon Negative.png) are provided for extremely constrained use cases where space limitations make the full wordmark impractical—specifically for profile pictures, avatars, and social media account images where dimensions are severely limited.

Both icons feature the letters "CC" (CNX Concierge) in a compact, legible format designed for recognition at small sizes. Icon Positive displays "CC" in CNX Navy on transparent/light backgrounds, while Icon Negative displays "CC" in white on dark backgrounds.

Critical Rule:

Icons are never used in general brand applications. They are reserved exclusively for profile pictures and avatars where the full wordmark cannot fit. The wordmark remains the primary brand identifier in all other contexts.

Example: LinkedIn, Twitter, Facebook profile pictures where the full wordmark would be illegible at small sizes. The "CC" icon provides clear brand recognition in constrained spaces.

Social media platforms require square or circular profile images. Use visual/png/Icon Positive.png for light backgrounds or visual/png/Icon Negative.png for dark backgrounds, with rounded corners (border-radius: 50%), sized appropriately for each platform's requirements.

Prohibited Uses

Never Use Icons For:

  • Website headers or navigation
  • Business cards or letterhead
  • Banners or marketing materials
  • Print collateral
  • Email signatures
  • Document headers or footers
  • Any application where the full wordmark can be used

Icon vs. Wordmark Decision

  • Use Icon: Only when space is 64px or smaller (profile pictures, avatars, favicons)
  • Use Wordmark: All other applications, regardless of size constraints
  • Rule of thumb: If the wordmark is legible, use it. Icons are a last resort.
