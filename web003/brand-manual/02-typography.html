Typography

Typography 4: Trirong — Traditional Thai Serif Authority

Trirong is a serif typeface with native Thai script support, featuring a narrow, tall structure reminiscent of traditional Thai typography. It combines thick and thin strokes with oval-shaped looped terminals, creating a formal, institutional appearance that honors Thai typographic heritage while maintaining professional gravitas.

Wordmark Rules

  • Always use ALL CAPS
  • Letter-spacing: 0.1em
  • Font weight: 700 (Bold)
  • Color: CNX Navy (#0c2b5c) on light backgrounds
  • Color: White (#ffffff) on dark backgrounds
  • No symbols or icons in wordmark
  • Typography-only identity

Text Specifications

Minimum Sizes

  • Web: 16px (1rem) minimum for body text
  • Print: 10pt minimum for body text
  • Captions: 0.9rem (14.4px) minimum

Line Height

  • Body text: 1.7 (optimal readability)
  • Headings: 1.2–1.4 (tighter for impact)
  • Captions: 1.6 (slightly tighter)

Line Length

  • Maximum: 70–80 characters per line
  • Optimal: 60–70 characters
  • Prevents eye strain and improves readability

Spacing

  • Paragraph spacing: 1rem minimum
  • Section spacing: 3–4rem
  • Letter-spacing: Normal (except wordmark)

Typography Rules

Do

  • Use Trirong for all brand communications
  • Maintain consistent hierarchy
  • Respect minimum font sizes
  • Use appropriate line heights
  • Limit line length to 70–80 characters
  • Maintain generous whitespace

Don't

  • Mix with other typefaces (except system fallbacks)
  • Use decorative or display fonts
  • Compress or extend letterforms
  • Use excessive bold or italic
  • Reduce font sizes below minimums
  • Crowd text with insufficient spacing
