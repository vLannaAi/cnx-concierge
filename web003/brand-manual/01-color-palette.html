Color Palette

Palette 2: CNX Deep Navy — Complete Specifications (Default/Approved)

The CNX Concierge color palette is built around CNX Navy as the primary brand color. This navy-led system positions the brand alongside private banks and advisory practices, conveying institutional credibility while maintaining neutrality and restraint.

Note: Palette 2 (CNX Deep Navy) is the default and approved palette for all CNX Concierge materials. This palette is used in the approved landing page and should be used as the primary reference for all brand applications.

Color Swatches

CNX Navy

#0c2b5c

Primary brand color. Used for the CNX Concierge wordmark, primary navigation, and key headings.

CNX Light Paper

#f7f8fb

Soft, cool background for sections and panels. Keeps navy feeling grounded, not heavy.

Harbor Gray

#59647a

Secondary text and dividers. Provides depth without introducing a new hue family.

Lavender Accent

#76235f

Reserved for links, small callouts, and subtle UI elements. Adds subtle sophistication without competing with navy.

Ink

#1a1a1a

Body text on light backgrounds. Provides maximum readability for long-form content.

White

#ffffff

Cards and elevated content. Pure white for maximum contrast when needed.

Color Application Criteria

Foreground Color

Primary

CNX Navy (#0c2b5c) — Used for wordmark and key headings. This is the primary brand signal. All major headings (H1, H2) should use CNX Navy to establish brand presence.

Body Text

Ink (#1a1a1a) — Used for body text on light backgrounds. Provides maximum readability for long-form content. Never use CNX Navy for body text as it reduces readability.

Secondary

Harbor Gray (#59647a) — Used for captions, metadata, labels, and secondary information. Supports hierarchy without competing with primary text.

Accent

Lavender (#76235f) — Used for links and interactive elements. Provides visual interest while maintaining sophistication.

Background Color

Primary

CNX Light Paper (#f7f8fb) — Main background for content areas. Soft, cool tone that keeps navy feeling grounded. Used for page backgrounds and section containers.

Elevated Content

White (#ffffff) — Used for cards, panels, and elevated content. Provides maximum contrast and clean appearance for important information.

Negative Application

CNX Navy (#0c2b5c) — Used for negative/reverse applications where text appears on dark backgrounds. See Negative Application section for detailed guidelines.

Accent Color

Usage Rules

Lavender Accent (#76235f) must be used sparingly:

  • Links and hover states
  • Small callouts and highlights
  • Subtle UI elements (buttons, icons)
  • Maximum 10% of visual space
  • Never used as background flood
  • Never used for large text blocks

Purpose

The accent color adds subtle sophistication without competing with the primary navy. It signals interactivity and draws attention to actionable elements without overwhelming the restrained palette.
