## Optimization & Production Notes

### File Format Recommendations

#### Web Use
- **Current Format:** PNG (lossless, large file sizes)
- **Recommended:** WebP with PNG fallback
  - 30-50% smaller file sizes
  - Maintains quality
  - Better performance
- **Alternative:** Optimized JPEG for photographs (if transparency not needed)

#### Print Use
- **Current Format:** PNG (acceptable)
- **Recommended:** 
  - High-resolution PNG for images with transparency
  - High-quality JPEG (95-100% quality) for photographs
  - CMYK color space for print production

### File Size Optimization

#### Building Images
- **Current:** 1.5-1.9 MB each
- **Target (Web):** 200-500 KB per image (WebP format)
- **Target (Print):** Maintain high resolution, optimize color space

#### Founder Images
- **Current:** 126 KB - 2.1 MB (inconsistent, placeholders to be replaced)
- **Target (Web):** 100-300 KB per image (WebP format, optimized for performance)
- **Target (Print):** Minimum 1000 × 1400 pixels, high quality
- **Performance Note:** Heavy media files can slow page loading — prioritize optimization

### Responsive Image Strategy

#### Building Images
Create multiple sizes for responsive design:
- **Desktop:** 1496 × 997 (or original)
- **Tablet:** 1024 × 683
- **Mobile:** 768 × 512
- **Thumbnail:** 400 × 267

#### Founder Images
Create multiple sizes:
- **Large:** 1039 × 1428 (or higher)
- **Medium:** 600 × 900
- **Small:** 300 × 450
- **Thumbnail:** 150 × 225

### Color Grading Recommendations

Per brand guidelines, all images should have:
- **Muted tones:** Reduce saturation by 10-20%
- **Warm undertones:** Slight shift toward warm (yellow/orange)
- **Desaturated look:** Avoid vibrant colors
- **Quiet mood:** Soft lighting, contemplative composition

**Action Required:** Review all images for color grading consistency
